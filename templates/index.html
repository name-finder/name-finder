Predict Age for Single Name

Usage:

  • Send a GET request to the /predict-age endpoint
  • Field mid_percentile refers to the middle X% of individuals with the name

GET request:

{ "name": "Alexander", "sex": "M", "mid_percentile": 0.8 }

Response:

{ "params": { "name": "Alexander", "sex": "m", "mid_percentile": 0.8 }, "lower": { "percentile": 0.1, "year": 1981, "age": 44 }, "upper": { "percentile": 0.9, "year": 2016, "age": 9 } }


Predict Gender

Usage:

  • Send a POST request to the /predict-gender endpoint
  • Format data as JSON string
  • Use Content-Type as application/json

POST request:

{ "data": [ {"name": "Gregory"}, {"name": "Anne-Marie"}, {"name": "cAsEy"}, {"name": "Super-Uncommon-Name"} ], "after": 1950, "before": 2000 }

Response:

[ { "name": "Gregory", "matched_name": "Gregory", "gender_prediction": "m", "f_pct": 0.0, "m_pct": 100.0 }, { "name": "Anne-Marie", "matched_name": "Annemarie", "gender_prediction": "f", "f_pct": 100.0, "m_pct": 0.0 }, { "name": "cAsEy", "matched_name": "Casey", "gender_prediction": "x", "f_pct": 41.0, "m_pct": 59.0 }, { "name": "Super-Uncommon-Name", "matched_name": "Superuncommonname", "gender_prediction": "unk", "f_pct": NaN, "m_pct": NaN } ]
